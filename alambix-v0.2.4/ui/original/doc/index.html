Documentation Alambix

Introduction

Alambix est un challenge de programmation multi-tâches (version Linux).

Cette documentation concerne les instructions à suivre pour programmer l'Alambix. Plusieurs étapes vous sont proposées pour relever le défi.

En complément, une documentation sur l'interface de programmation Alambix (API) est également disponible. Cette documentation est qualifiée de documentation de l'API dans la suite de ce document.

Animer l'Alambix avec des threads

Quatre personnages vont agir et interagir dans l'Alambix :

  • client0 et client1, les deux clients de l'Alambix ;
  • waiter, le serveur de l'Alambix ;
  • bartender, le barman de l'Alambix.

À chacun de ces personnages correspond un thread de programmation (cf. la documentation de l'API The Alambix pthreads).

La création des threads des personnages et leur lancement (ou démarrage) se feront suite à un clique sur le bouton Start de l'interface utilisateur (cf. The Alambix user interface de la documentation de l'API). Pour cela une fonction alambix_start() doit être implémentée (cf. Start the Alambix de la documentation de l'API). Cette fonction sera appelée suite au clique sur le bouton Start.

Remarque : il vous sera demandé de sélectionner au moins une boisson pour chaque client de l'Alambix avant d'animer les personnages (bouton Start). Un clique sur un client fait apparaître un menu de sélection des boissons que souhaite commander le client.

À partir de là vous pouvez tenter d'animer l'Alambix...
