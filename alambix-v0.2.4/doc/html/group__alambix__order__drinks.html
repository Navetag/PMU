Detailed Description

Functions to place an order.

In order to place an order, a client may follow an algorithm like the following:

// order all the client's chosen drinks
char * drink;
while ((drink = alambix_choose_drink()) != NULL)
{
alambix_order_drink(drink);
}

void alambix_choose_drink ( )

Choose a drink.

Get a drink chosen by an Alambix client (that will be ordered later).
This function must be called by an Alambix client thread (either alambix_client0_thread or alambix_client1_thread).

Returns
The next drink that has been chosen by the client, null at the end of the chosen drinks set.
