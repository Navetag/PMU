The Alambix documentation

The Alambix documentation is divided into two parts:

  1. The programming documentation, that describes the way to program the Alambix.
  2. The instructions documentation, that describes what you need to do to program the Alambix.

The current documentation (that you are reading) is the main entry point to access the programming documentation. It explains how to access the instructions documentation.

The programming documentation

You can find the documentation of the Application Programming Interface (Alambix API) in the Modules section (second tab "Modules" just above).

The Modules section also supplies information on how to compile an Alambix program.

The instructions documentation

The instructions to fully program the Alambix is also called Help. It is given in a document that you will access through the Alambix user interface (described hereafter).

Step 1: launch the instructions documentation (see below) explains how to program the Alambix in order to access that documentation.

The Alambix user interface

When you launch the Alambix the following user interface (UI) is launched:

The left part of the UI gives information on the Alambix program (memory and CPU usage, process ID, etc.), and provides:

  • a Help button, to launch the instructions documentation (as described below),
  • a Start button (its usage is described in the instructions documentation),
  • a Quit button, to quit the Alambix at any time.

The right part of the UI corresponds to the scene of the Alambix.
